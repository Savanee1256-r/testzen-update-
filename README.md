# TestZen

Modern web app for exam management. Teachers create exams, students take them, results tracked.

## Quick Start

```bash
npm run install:all
npm run dev:frontend
```

- Frontend: http://localhost:5173
- Backend: http://localhost:3000 (auto-proxied)

## Scripts
- `npm run dev:frontend` - Start frontend dev server
- `npm run dev:backend` - Start backend
- `npm run build` - Build frontend
- `npm run start:webapp` - Build + start production

PWA-ready with TestZen icon/name.
